
# 我不要當主角 - 遊戲原型 🎮

這是一款橫向躲避型遊戲，主角是一位內心厭世的上班族，任務是：**盡可能地不與他人攀談**。

## 🕹️ 操作方式

- 使用方向鍵 ↑ ↓ 移動（模擬上滑）
- 碰到 NPC：扣能量
- 撿起黃色物品：回血
- 抵達一定距離即過關

## 📁 檔案結構

```
not-the-main-character/
├── index.html
├── game.html
├── result.html
├── style.css
├── script.js
└── assets/
```

## 📡 部署說明

可直接部署於 GitHub Pages。打開 `index.html` 即可遊玩。
